# X-Go

- 分布式部署或者集群部署
- 任务队列精准控制
- 二进制文件或Docker直接部署
- 欢迎有需求的老板PM

## 技术栈
### 前端
Vue3+Vite+Naitive UI

### Server端
Go+Beego-Api+Redis+Mysql+RPCX

### Worker

Go+Redis+Mysql
